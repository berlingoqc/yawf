Welcome to the yase project

A modular framework for experimenting with 3D graphics

Goal of the project

This project start of as my final assignement in my entry course on 3D programming with OpengGL

The initial goal where the following :

  • Manage all the file of a interactive 3D environment
  • Manage the reading, allocation and deallocation of the assets
  • Create a software to import asset and create 3D environment
  • Create a library to use the environment to create clips or small game

But things got more complex when i discovered Emscripten.
I decide to make my application work on the browser with asset serve by a web server

  • Responsive design
  • Cross-everything
  • Documentation
  • jQuery plugins
